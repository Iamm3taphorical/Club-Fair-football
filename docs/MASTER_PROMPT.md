# Master Prompt Summary

FootballVerse AI must be a cross-platform localhost web application, not a desktop app.

Primary interface:

```text
http://localhost:8501
```

The app must feel like a professional football platform inspired by EA FC, Football Manager, Champions League broadcast graphics, Premier League broadcast graphics, and modern football analytics products.

It must include:

- Player Mode
- Coach Mode
- Football DNA
- Browser penalty game
- Tactical board
- Commentary
- Report cards
- Leaderboard
- Prize/reward system
- Demo Mode

The MVP frontend is Streamlit, with a future upgrade path to React + FastAPI.
